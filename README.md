# Project 1: Rule-Based AI Chatbot

## Description

A simple AI chatbot developed using Python that responds to user queries based on predefined rules and responses. The chatbot accepts user input, processes it, and returns the appropriate response from a predefined knowledge base.

## Features

* Interactive command-line chatbot
* Predefined question-response system
* Handles greetings and common conversations
* Exit commands support (exit, quit, bye)
* Lightweight and beginner-friendly AI project

## Technologies Used

* Python

## Workflow

User Input → Text Processing → Rule Matching → Response Generation

## Sample Interaction

You: hello

Bot: Hi! How can I help you?

You: how are you

Bot: I am doing great!

You: bye

Bot: Goodbye!

## Learning Outcomes

* Python programming fundamentals
* Conditional logic
* Dictionary data structures
* User input handling
* Basic conversational AI concepts

# Project 2: Smart OCR and Object Detection System

## Description

A Computer Vision project that combines Optical Character Recognition (OCR) and Object Detection. The system can extract text from images and identify objects present in the same image using deep learning techniques.

## Features

* Image preprocessing
* Text extraction using Tesseract OCR
* Object detection using MobileNet SSD
* Bounding box visualization
* Confidence score generation
* Annotated image output
* Statistics and reporting

## Technologies Used

* Python
* OpenCV
* Tesseract OCR
* NumPy
* Pandas
* MobileNet SSD

## Workflow

Input Image

↓

Image Preprocessing

↓

OCR Text Extraction

↓

Object Detection

↓

Confidence Analysis

↓

Output Generation
