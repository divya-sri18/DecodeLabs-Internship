## Project 2: Smart OCR and Object Detection System

### Overview

A computer vision project that combines Optical Character Recognition (OCR) and Object Detection to extract text and identify objects from images.

### Features

* Text extraction from images using Tesseract OCR
* Image preprocessing using OpenCV
* Object detection using MobileNet SSD
* Bounding box visualization
* Confidence score display
* Output image generation

### Technologies Used

* Python
* OpenCV
* Tesseract OCR
* NumPy
* Pandas
* MobileNet SSD

### Workflow

Input Image → Preprocessing → OCR → Object Detection → Output Generation
